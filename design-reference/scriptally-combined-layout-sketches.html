My lean

Sketch 1's shell, with Sketch 2 as its true-empty fallback. The sidebar-plus-workspace is the structure that earns its keep — materials always visible, packages front and centre, the build-from relationship made obvious, and no empty tab to switch to. Seed it (sketch 1) so the very first impression is a working, premium page; if a user clears the examples, fall back to the how-it-works + templates empty state (sketch 2) rather than blank cards. Sketch 3 is the one to pick instead if you'd rather the active package always be the hero and treat materials as a lighter companion column — but the sidebar version makes materials feel like a first-class half of the feature, which suits "this is a real Pro tool."
